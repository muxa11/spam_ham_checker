# spam_ham_checker
The project takes an email, sms and checks whether it is sapm or ham(not spam)
The Emails are saved in emails folder and named as s_no.YYYY-MM-DD."farmer" or "GP"."ham" or "spam".txt
The dataset can be downloaded from http://www2.aueb.gr/users/ion/data/enron-spam/preprocessed/enron1.tar.gz
